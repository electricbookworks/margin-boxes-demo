---
title: Margin boxes demo
---

```
@page chapter {
	background-color: purple;
	border: 2pt solid cyan;
	margin-top: 3em;
	margin-inside: 20mm;
	margin-bottom: 5em;
	margin-outside: 30mm;
    @top-left-corner {
        content: "this is the top left";
        background: green;
        padding: 0;
        margin: 0;
        border: 1pt solid blue;
    }
    @top-center {
        content: "this is the top center";
        background: red;
        padding: 0;
        margin: 1em 0 0 0;
        border: 1pt solid blue;
        vertical-align: top;
    }
    @top-right-corner {
        content: "this is the top right";
        background: green;
        padding: 0;
        margin: 0;
        border: 1pt solid blue;
    }
    @bottom-right-corner {
        content: "this is the bottom right";
        background: red;
        padding: 0;
        margin: 0;
        border: 1pt solid blue;
    }
    @bottom-center {
        content: "this is the bottom center";
        background: green;
        padding: 0;
        margin: 0 0 2em 0;
        border: 1pt solid blue;
        vertical-align: bottom;
    }
    @bottom-left-corner {
        content: "this is the bottom left";
        background: red;
        padding: 0;
        margin: 0;
        border: 1pt solid blue;
    }
}
```
